# CodeAlpha Advanced Chatbot

Advanced Python Chatbot with greeting, date-time, calculator, and help features.

## Features
- Greeting System
- Bot Information
- Current Date and Time
- Calculator Operations
- Help Menu
- Error Handling

## Technologies Used
- Python

## How to Run
1. Open chatbot.py
2. Run the file
3. Enter commands such as:
   - hello
   - name
   - time
   - date
   - add
   - help
   - bye
